## 课堂Tip

### linux命令

```
ls // 查看所有文件（不包含隐藏文件，隐藏文件即以.开头的文件）
ls -la // 查看所有文件（包含隐藏文件）
cat 文件名 // 查看文件内容
pwd // 当前的工作目录的路径
```

## NVM: Node Version Manager（Node版本管理）

可用于在不同node版本间切换。

**nvm常用命令：**

```
nvm list // 列举已安装的node的版本
nvm use 版本号 // 切换到某个版本
```
